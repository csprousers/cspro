CSPro Licenses

The Census and Survey Processing System (CSPro) is designed by the U.S. Census Bureau, ICF, and Serpro S.A., with funding from the U.S. Agency for International Development. The following programs are used to develop the software:

  • Microsoft Visual Studio Professional (C, C#, and C++)
  • Android Studio (C, C++, Java, Kotlin)
  • NSIS: Nullsoft Scriptable Install System (install package)
  • wkhtmltopdf (help documentation)

The software uses several open-source software packages, the licenses of which follow the CSPro license.
